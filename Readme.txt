######## Getting started #########

Copy the base folder 
rename base folder with project name
cd to project name
rename replace folder with project name
in Procfile replace 'replace' with project name
cd to project name (inside one)
in settings.py replace 'replace' with project name
in wsgi.py replace 'replace' with project name
done

##################################


######## Make an app #############

cd to project folder
in settings.py install the app
in urls.py add urlpattern
done

######## Make an app work #######

cd to app folder
create urls.py
import views
in urls.py add url pattern using views
in views.py def func
done
